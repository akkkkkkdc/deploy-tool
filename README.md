# 🚀 Jar包一键部署工具

一个带图形界面的 Windows 桌面应用，用于一键完成：Maven 打包 → 上传 Jar 到 Linux 服务器 → 执行启动脚本。

## 功能特性

- ✅ **服务器管理**：增删改查，密码加密存储
- ✅ **应用管理**：每个应用绑定一个服务器，配置 Jar 名和启动脚本
- ✅ **一键部署**：选中应用 → 点击部署 → 全自动完成三步
- ✅ **实时日志**：部署过程实时显示
- ✅ **深色主题**：护眼界面
- ✅ **免密码**：配置一次，以后不用输密码

## 下载 exe（最方便）

点击 GitHub 右侧 **Releases** → 下载最新的 `部署工具.exe` 即可，双击运行，无需 Python 环境。

> 💡 Releases 下载地址：https://github.com/akkkkkkdc/deploy-tool/releases

## 使用方法

### 方式一：直接运行 exe（推荐）

下载 Releases 里的 exe，双击运行，无需安装任何依赖。

### 方式二：运行源码（需要 Python 环境）

```bash
pip install -r requirements.txt
python main.py
```

### 方式三：自行打包 exe

双击运行 `package.bat`，会在 `dist/` 目录下生成 `Jar包一键部署工具.exe`。

## 依赖（源码运行才需要）

- Python 3.9+
- PyQt6
- paramiko
- cryptography
- PyInstaller（仅打包用）
- Maven（系统安装，部署时会自动调用）
- SSH（系统自带或装 Git 时带的 ssh）

## 使用步骤

1. **添加服务器**：点击左侧「+ 服务器」，填写 IP、用户名、密码、项目路径
2. **添加应用**：点击左侧「+ 应用」，填写应用名、Jar 包名、启动脚本名，选择所属服务器
3. **一键部署**：点击「🚀 部署」，等待完成即可～

## 常见问题

**Q：部署时提示 `mvn` 不是命令？**
> 确保系统 PATH 里加了 Maven 的 bin 目录，或者在 IDEA 的 Terminal 里运行 exe（通常 IDEA Terminal 已配置好 Maven）

**Q：密码会泄露吗？**
> 不会。密码用 AES 加密存在本地数据库里，不会上传任何数据

**Q：可以同时部署到多台服务器吗？**
> 当前版本每次部署一台，未来可扩展
