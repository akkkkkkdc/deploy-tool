# 荆铸一键部署工具 - 设计规范

## 核心功能

### 1. 服务器管理（增删改查）
- 服务器名称（自定义别名）
- IP地址
- 用户名
- 密码（加密存储）
- 服务器项目路径（如 `/home/wuyuan/server/jingzhu-imaster`）
- 备注

### 2. 应用管理（增删改查）
- 应用名称
- Jar包名（如 `industry-0.0.1-SNAPSHOT.jar`）
- 启动脚本名（如 `uu-industry.sh`）
- 关联的服务器（一个应用对应一个服务器）
- Maven模块名（非必填，用于多模块项目）

### 3. 一键部署
选中应用 → 点击部署 → 自动完成：
1. 本地 `mvn clean package`
2. scp 上传 jar 到服务器
3. ssh 执行启动脚本
4. 实时显示部署日志

### 4. 批量部署（进阶）
一个应用部署到多台服务器

### 5. 部署历史记录
记录每次部署的时间、状态、日志

---

## 技术方案

### 界面：PyQt6 + CustomTkinter
- 现代扁平风格，深色主题
- 左侧：服务器列表 + 应用列表（树形结构）
- 右侧：部署控制台（实时日志）
- 底部：状态栏

### 数据存储：SQLite
本地数据库存储服务器和应用配置，密码加密存储。

### 部署执行：PyLauncher
调用本地 maven / scp / ssh，不依赖额外工具（用 paramiko 做 SFTP，取代 sshpass）

### 打包：PyInstaller
生成单个 Windows exe，可安装到桌面
